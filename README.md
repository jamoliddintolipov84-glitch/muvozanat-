# 🪐 Muvozanat Bot

**Muvozanat** — intizom, moliya va sog'lom turmush tarzini muvofiqlashtiruvchi Telegram bot. 
Bu loyiha foydalanuvchilarga o'z vaqtini to'g'ri boshqarish va hayotiy balansni topishda yordam beradi.

## ✨ Xususiyatlari:
* 🤖 **AI Coach:** Foydalanuvchi bilan muloqot qiluvchi aqlli yordamchi.
* 👥 **Referral Tizimi:** Botdan to'liq foydalanish uchun do'stlarni taklif qilish mexanikasi.
* 💾 **Doimiy Baza:** Ma'lumotlar SQLite bazasida xavfsiz saqlanadi.
* 🚀 **Tezkor Ulashish:** Bir bosishda do'stlar bilan ulashish (Share URL).

## 🛠 Texnologiyalar:
* **Python** (aiogram 3.x)
* **SQLite** (Ma'lumotlar bazasi)
* **Render** (Hosting)

## 🚀 Ishga tushirish:
1. `pip install -r requirements.txt`
2. `python bot.py`
